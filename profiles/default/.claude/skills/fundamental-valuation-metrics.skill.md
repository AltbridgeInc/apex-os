# Valuation Metrics Analysis Skill

This skill enables comprehensive valuation assessment using both relative and absolute methods to determine if stocks are undervalued, fairly valued, or overvalued.

**Valuation Methods Applied**:
- Relative: P/E, PEG, P/S, EV/EBITDA, P/B
- Absolute: DCF, DDM
- Growth-adjusted: Rule of 40 (SaaS)

**Auto-loaded for**: fundamental-analyst agent

**Scoring Output**: 0-10 valuation score based on:
- Multiple metrics comparison
- Peer relative valuation
- Historical valuation ranges
- DCF-derived fair value
- Margin of safety

**Contextual Adjustments**:
- Market conditions (bull/bear)
- Growth stage (high/moderate/mature)
- Quality premium (moat strength)

Agents with this skill automatically assess whether current price offers attractive entry point relative to intrinsic value.
